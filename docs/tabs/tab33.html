★ Key features

  • Enter any 15 or 18-character Salesforce record ID to query sharing
  • Shows access level (All, Edit, Read) for each user/group/role entry
  • Reveals the sharing reason: Owner, Rule, Manual, ImplicitChild, etc.
  • Displays both direct group membership and the related group name
  • One-click "OpenUrl" to jump directly to the record's Sharing Settings page in Salesforce
  • Works on any standard or custom Salesforce object with a sharing model
