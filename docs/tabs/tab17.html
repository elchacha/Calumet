★ Key features

  • Search users by name or username with autocomplete — results automatically switch to the Search sub-tab
  • Double-click any user to generate a Salesforce impersonation URL and open it instantly in Chrome
  • Choose the Chrome profile (Private, or any named profile) for each login session
  • Optional Target Id — navigate directly to a specific record after impersonation
  • Mode console — opens the Lightning App Builder console URL instead of the standard home page
  • CommunityLogin — uses the Experience Cloud (community) login endpoint for external users
  • Automatic debug trace flag creation for the impersonated user (configurable duration in minutes)
  • Favorites system — right-click to bookmark frequent users; favorites persist across sessions per org
  • Per-favorite Chrome profile — assign a different Chrome profile to each favorite user
