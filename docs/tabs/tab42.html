★ Key features

  • Analyze all Lightning pages across profiles and detect legacy (non-LWC) components via the FD score
  • Color-coded FD/CL badges instantly identify problematic pages (green = clean, red = critical)
  • Layout Dependencies panel shows which classic layouts a FlexiPage still relies on, updated on row selection
  • Detect unused FlexiPages never assigned to any app or profile — safe cleanup candidates
  • Double-click any row to open the Lightning page directly in Salesforce Setup
  • Filterable by AppLabel, Name, Object, or Profile for quick drill-down across large orgs
