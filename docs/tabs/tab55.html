★ Key features

  • Loads all custom fields for any Salesforce object via autocomplete selection
  • Runs a parallel multi-threaded analysis (up to 25 threads) across all records to count how many populate each field
  • Displays Usage count, Percent, and composite Score badges color-coded from green (full) to red (empty)
  • Type badges visually distinguish text, number, date, boolean, picklist, reference, and datetime fields
  • Single-click a picklist field to reveal per-value usage with active/inactive row coloring
  • Single-click any non-picklist field to see usage broken down by RecordType
  • Double-click any field row to open its setup page directly in Salesforce
  • Filter the entire analysis by a specific RecordType via the dropdown (picklist fields hidden in filtered view)
  • Analysis results are cached locally; re-running prompts for confirmation before overwriting
  • Export the full analysis to Excel for offline review and sharing
