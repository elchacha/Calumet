★ Key features

  • Multi-type search — a single search term is matched against all selected metadata categories simultaneously.
  • 14 supported metadata types — Custom Labels, Apex Classes & Triggers, VisualForce Pages, Aura Components, LWC, Custom Metadata, Validation Rules, Workflow Rules, Workflow Field Updates, Flows, Email Templates, FlexiPages, Lookup Filters.
  • Deep Apex search — when "Apex" is checked, the full class/trigger body is searched (not just the name), so you find every reference in the source code.
  • Result tabs appear only when matches exist — each result category shows its count in the tab header (e.g. Classes(7)). Empty categories are hidden automatically.
  • Flow filter bar — the Flows result tab includes a live filter input to narrow results by label, API name, type or status (semicolon-separated terms).
  • Double-click to open — double-clicking an Apex row opens the source in your local editor; clicking a Salesforce record row navigates to it in the browser.
  • Dependency analysis — the Dependency sub-tab uses the MetadataComponentDependency Tooling API to show all metadata items that use or are used by a selected element, with direct navigation on double-click.
  • Package XML integration — right-click any row to add the selected metadata to a deployment package.
  • Cache support — results are cached per org; tick "Clear cache" to force a fresh API extraction.
