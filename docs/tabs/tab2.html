★ Key features

  • Retrieve Salesforce metadata from any org using a package.xml manifest via the Metadata API
  • Interactive tree view of package contents — expand, collapse, remove individual members; missing files highlighted in red
  • Detect and retrieve Profile changes (field-level security, object permissions, Apex class access) since last retrieve
  • Detect and retrieve Layout Assignment changes automatically added to profiles during retrieve
  • API Version scanner — lists Apex classes and triggers with an API version more than 5 releases behind the current org; supports batch update to current version
  • Prepare Deployment — copy the retrieve folder to another org's deploy folder in one click, with optional folder clean and automatic tab navigation
  • Optional VS Code / SFDX workspace push after retrieve (converts metadata format)
  • Save retrieve packages as named ZIP files for later reuse; merge multiple zipped package.xml files
  • Dependency check — detect cross-component dependencies before deploying
