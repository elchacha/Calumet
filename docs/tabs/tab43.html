★ Key features

  • Scans every SObject that supports Record Types and enumerates all active Record Types via the Metadata API.
  • Queries record counts per Record Type using the Salesforce REST API and computes the usage percentage over the total for each object.
  • Shows Last Creation and Last Action dates so unused or stale Record Types can be easily spotted.
  • Extraction result is persisted to a local cache; a subsequent Read button loads it instantly without another API call.
  • Cache date is displayed next to the Read button so you know how fresh the data is.
  • Live text filter on the table (debounced, 500 ms) for fast lookup across all columns.
  • One-click Export to xlsx to share results with stakeholders, with optional auto-open after export.
