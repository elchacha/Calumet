★ Key features

  • Assignment matrix — profiles and permission sets as rows, custom apps as columns; checkmarks show current assignments at a glance.
  • Inline editing — click any cell to toggle an assignment; pending changes are highlighted in yellow (add) or red (remove) before being committed.
  • Batch column operations — each app column header has a 3-state checkbox: click once to remove all assignments, again to assign all, again to restore originals.
  • Row checkbox filter — tick one or more profile rows to focus columns; only apps relevant to the selected profiles remain visible.
  • Hide empty columns — automatically hides app columns where no profile has an assignment, reducing visual noise.
  • System Admin toggle — optionally show or hide the System Administrator profile row.
  • Pending change summary — a live counter shows how many assignments are queued to be added or removed before applying.
  • Apply / Cancel — a single click applies all pending changes to Salesforce via the API, or discards them entirely.
  • Export to Excel — exports the full matrix (all profiles x all apps) to an Excel file for offline review or audit.
  • First-column filter — type any text to instantly filter profile/permission set rows by name (300 ms debounce).
