★ Key features

  • Evaluate any Salesforce formula against a real record ID without deploying

  • Explode mode splits a compound formula on && into individual sub-conditions
  • Expand tab pretty-prints any raw formula into indented, readable structure
  • Optional SOQL query filter to narrow context before evaluation
