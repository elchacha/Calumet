★ Key features

  • Fetches all Salesforce governor limits in one click via the Limits REST endpoint
  • Displays used, remaining, and maximum values for every limit category
  • Shows a percentage-used figure to instantly identify limits under pressure
  • Classifies limits by type (Platform, Streaming, Analytics, Storage, Email, Workflows, PE-CDC) for easy scanning
  • Shows the reset period (Daily, Hourly, Monthly) so you know when headroom resets
  • "Hide unused" checkbox filters out limits with 0% usage, reducing noise
  • Refreshes the global API call-out counter in the main window status bar after each fetch
