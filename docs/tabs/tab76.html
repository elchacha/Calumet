★ Key features

  • Four sub-tabs: Users, Groups, Queue, and Others (all group types excluding Regular and Queue)
  • Users table with live filter on any column (name, username, profile, role)
  • "Active only" checkbox hides inactive users in the Users panel
  • Groups and Queue panels use a TreeTable showing parent/child hierarchy with expand/collapse
  • Others panel includes type-filter pills (Organization, Role, GuestUserGroup, PRMOrganization…)
  • Member count per group/queue displayed in the Users column for instant capacity review
  • All panels support Refresh to bypass cache and reload from Salesforce in real-time
