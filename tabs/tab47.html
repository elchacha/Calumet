★ Key features

  • Global view — one row per connected app showing aggregated usage count, distinct user count, admin-controlled flag, and description
  • Detailed view — one row per user × app combination, showing the individual token creation date, last used date, profile, and usage count
  • Quickly spot stale integrations: apps with zero usage or tokens not used in months
  • Identify over-permissioned apps via the AdminControlled column — apps not admin-controlled can be self-authorised by any user
  • Revoke tokens directly from the Detailed view: select one or more rows and click "revoke selected token" to cut access immediately without going to Setup
  • Filter by typing in the filter field (semicolon-separated terms for AND logic) — the record counter updates in real time
  • Refreshes the org's API callout limit counter after each extract
