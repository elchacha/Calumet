★ Key features

  • Three-tier recommendation engine — classifies every user as Safe to Deactivate, Review, or Keep based on login frequency, API calls, owned records, and role assignments.
  • Configurable inactivity threshold — set the number of days of inactivity that triggers the "Safe to Deactivate" flag (default 90 days).
  • Status filter — scope the analysis to Active users, Inactive users, or all users in the org.
  • Real-time checkbox filtering — toggle each recommendation category on/off instantly without re-querying Salesforce.
  • License savings estimate — the summary bar shows the immediate savings percentage if all "Safe to Deactivate" active users were deactivated.
  • Color-coded rows — blue for "Safe to Deactivate", orange for "Review", green for "Keep", matching the recommendation badge colours for at-a-glance scanning.
  • Activity & Decision columns — human-readable explanation of what signals drove the recommendation and what action to take.
  • Excel export — export the currently filtered table to Excel for licence review meetings or change requests.
  • Local cache — results are serialised to disk; re-opening the tab reloads instantly from cache with the analysis timestamp displayed.
