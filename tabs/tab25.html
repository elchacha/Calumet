★ Key features

  • Daily record-count extraction for up to 126+ Salesforce objects simultaneously
  • Date-range picker to extract only the days needed, avoiding full re-extraction
  • Persistent CSV storage per day: evolStorage/YYYY-MM-DD.csv
  • Interactive Chart.js line chart with per-object toggle in the sidebar
  • "Group by N days" smoothing to reduce noise on long date ranges
  • Sidebar sorted by record total (highest first) or alphabetically
  • Pre-selected default objects saved per org, restored on next chart load
  • Progress feedback during extraction: remaining object count shown live
