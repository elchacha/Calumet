★ Key features

  • Lists all custom profiles and permission sets with creation date, author, and license type
  • Custom Only filter hides standard Salesforce profiles, focusing on org-specific ones
  • No active users filter surfaces profiles with zero active users — prime deletion candidates
  • Displays both Active and Inactive user counts per profile to judge the migration effort
  • Autocomplete replacement profile field locks in the target profile with a green border and a clear button
  • The delete selected profiles button is enabled only when the form is valid — replacement profile required if any users exist
  • After deletion, the tab auto-refreshes to show the updated profile list
  • Integrates with the Salesforce Partner / Metadata API for safe profile removal and user reassignment
