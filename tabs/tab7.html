★ Key features

  • Object Permission audit — view Create / Read / Edit / Delete / View All / Modify All for every Profile and Permission Set in a single table.
  • Security Type badges — Profile rows are highlighted in blue, Permission Set rows in purple, making it easy to distinguish them at a glance.
  • Smart filtering — filter by object name, toggle Profiles / Permission Sets on or off, and hide entries with no assigned users.
  • Sharing rules browser — list all sharing rules per object with their access level (Read / ReadWrite), criteria, sharing target, and last modified date.
  • Disk cache — results are cached per org to cache/object_access.cache and cache/sharings.cache, with cache date displayed in the toolbar.
  • Excel export — export the full Object Permission matrix to .xlsx for offline analysis or audit reporting.
  • User count column — the Users column shows how many users are assigned to each Profile / Permission Set, helping identify orphaned or over-provisioned entries.
