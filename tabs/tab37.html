★ Key features

  • Compare two profiles from the same org or across two different orgs
  • Autocomplete profile name fields with live suggestions from the org
  • Locked profile name fields with one-click X button to clear
  • Three result tabs: Diff (differences), Profile 1 (items missing in Profile 2), Profile 2 (items missing in Profile 1)
  • Tab counters show the number of exclusive items per profile
  • Filter diff results with multi-criteria text search (semicolon-separated)
  • Hide AppExchange namespaced items to focus on org-specific differences
  • Right-click context menu to export selected differences as a deployable profile XML
  • Cached profile XML files for faster re-comparison without re-fetching
