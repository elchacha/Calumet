★ Key features

  • Auto-detects the SObject type of any 15 or 18-character Salesforce record ID
  • Walks all child relationships and queries each object for the user's access level
  • Progress bar reports per-query completion (N / total queries)
  • User autocomplete locks the field once a user is selected, resolving their ID
  • "Show no access" toggle includes records with zero rights for full-coverage audits
  • Access levels displayed: Delete, Edit, Read, All, None — color-coded in the table
