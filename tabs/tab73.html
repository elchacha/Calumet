★ Key features

  • Add any number of users to a comparison basket using an auto-complete user search field.
  • Retrieve the full access profile for each user: permission sets, permission set groups, profile assignment, role and group memberships.
  • In single-user mode, all assignments are listed with type badges and direct/indirect membership indicators.
  • In compare mode (2+ users), only differences are shown — one column per user with a check (has it) or a cross (does not have it).
  • Colored badges distinguish PermSet, PermSetGroup and Profile in the Permission Sets tab, and Regular / Queue / Role group types in the Groups tab.
  • Role assignments are displayed per user below the basket — identical roles or diverging roles are both surfaced clearly.
  • The current connected user is automatically pre-filled in the search field when the tab opens.
  • Basket pills can be removed individually with the × button or cleared all at once.
  • Status label reports the exact count of perm set and group differences found after a compare run.
