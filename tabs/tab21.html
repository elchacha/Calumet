Process the log and shrink it to present an easy-to-read methods file. Generates a hierarchical call tree showing method durations.

How it works
1. Select a .log file from your log cache folder.
2. Calumet parses the log and builds a hierarchical call tree of all Apex methods.
3. The tree popup shows each method with its individual and cumulative duration in ms.
4. Double-clicking any row in the Display Logs tab triggers the same analysis automatically.

Input a log file with at least: Database (Info), Apex Code (Finest), Apex Profiling (Info), System (Fine). Detects and reports SOQL 101 governor limit violations with full callstack context.

Required log levels for SOQL 101 detection
Database: Info  |  Apex Code: Finest  |  Apex Profiling: Info  |  System: Fine

The analyser identifies every SOQL query executed inside a loop and highlights the callstack that led to each 101 error. Results are shown per trigger / class with the offending query text.

Select a log file to apply a filter and extract only the relevant lines. Useful for focusing on a specific class, USER_DEBUG output, or error category in a large log.

Filter capabilities
Filter by keyword, log category (USER_DEBUG, EXCEPTION_THROWN, DML_BEGIN, SOQL_EXECUTE_BEGIN…), class name, or log level. The filtered output is written to a new file alongside the original and opened automatically.

★ Key features

  • Search org debug logs with filters on user, status, minimum duration and date range
  • Color-coded rows: error / exception logs in red, success logs in white — at a glance triage
  • Right-click context menu: download, filter, analyse methods, detect 101 errors, open in VS Code
  • Double-click any row to instantly open a hierarchical method call tree with per-method timing
  • Recalculate real operation name by parsing log content (handles Batch Apex, QueueableHandler, /aura, etc.)
  • Manage trace flags: create / update / delete per user, Apex class, Apex trigger, or profile
  • Duration shortcuts: one-click 30mn / 1h / 2h / 8h / 24h expiry for trace flags
  • Real-time log tail monitor (Open Monitor) — stream new logs as they arrive in the org
  • Log Analyser: parse a local .log file into a collapsible method call tree
  • SOQL 101 Analyser: detect governor limit violations with full callstack and query text
  • Filter Log: extract specific lines from large log files by keyword or category
  • Local cache management: view size, open folder and clear cached log files
