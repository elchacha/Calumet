Tab 27

Extract and browse all picklist fields from any Salesforce object, including their allowed values per Record Type. Each picklist field is shown in its own sub-tab with a matrix view: rows are picklist values (API name + label), columns are Record Types. Default values are highlighted with a green badge. Results can be exported to Excel for offline review or change-management documentation.
