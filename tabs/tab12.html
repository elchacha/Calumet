SOQL Executor

Tab 12

A multi-tab interactive SOQL query editor with syntax highlighting, auto-complete, and live result display. Run queries against both the Partner API and Tooling API, apply real-time filters on results, export to CSV / Excel, and delete selected records directly from the result grid. Each query runs in its own closable tab; tabs are automatically restored from the previous session.

Key Features
Syntax-highlighted SOQL editor

Built on RichTextFX CodeArea. Keywords, functions, object name, resolved fields, and string literals are each coloured distinctly. Highlighting re-runs 500 ms after the user stops typing.

Ctrl+Space auto-complete

Pressing Ctrl+Space (or pausing mid-word) triggers a dropdown with all fields and relationship paths for the detected FROM object, sourced from the cached FieldHelper metadata.

Multi-tab queries

Each query lives in its own closable tab. Click "+" to open a new blank tab. On application exit, open queries are saved to a queryBackup file and restored on next launch.

Partner API & Tooling API

Toggle "Tooling" to switch between the standard Partner API (sObjects) and the Salesforce Tooling API (ApexClass, Flow, etc.). Syntax highlighting adapts to the selected API's schema.

Bulk Mode (API v2)

Enable "Bulk Mode" for large datasets. The header shows the daily Bulk V2 job quota (used/max). Results can still be exported from the CSV file produced by the bulk job.

SELECT * expansion

Queries starting with SELECT * are automatically expanded to all fields. The "Select * config" popup lets you define priority columns that appear first in the result grid.

Fast Count

Check "Fast Count" to run an approximate COUNT() instead of fetching all records. Useful for quickly gauging dataset sizes before running a full query.

Record deletion

Select one or more ID cells in the result grid and press Delete. A confirmation popup appears before the deletion is executed. Production orgs require explicit enablement in the global config.

Live filter

The filter bar (bottom of the result pane) narrows displayed rows in real time (500 ms debounce). Multiple terms can be combined with ";". The "Apply filter" checkbox re-applies the filter after each new query execution.

CSV / Excel export

Export results to CSV (configurable separator, optional double-quoting) or directly to .xls Excel format via Apache POI. For bulk-mode queries, the raw CSV file produced by the bulk job is exported instead.

Saved queries

Type a name in the combo box and click "Save query" to persist the current query to soqlSaved/<name>.sql. "Open query folder" reveals the folder in the OS file manager. "Refresh list" rescans saved files.

Record double-click navigation

Double-clicking a 15/18-character Salesforce ID opens a detail popup for that record. Right-double-click opens the record URL in the browser. Alt+click also triggers URL navigation.
