SOQL Executor

Tab 12

Multi-tab SOQL query editor with syntax highlighting, Partner API and Tooling API execution, inline result filtering, CSV/Excel export, and inline record deletion. Query tabs persist across sessions via a local backup file.

Key Features

  • ✎Syntax highlighting — RichTextFX CodeArea with keywords, field names, constants, and operators colored in real time.
  • ⇧Ctrl+Space autocomplete — field-level and object-level suggestions from the org describe cache.
  • 📃Multi-tab queries — open as many query tabs as needed; all queries are saved and restored across sessions.
  • 🔧Tooling API toggle — switch between Partner API and Tooling API per query tab.
  • ⚡Bulk Mode — use Bulk API v2 for high-volume queries (100k+ records) without hitting governor limits.
  • 📄Export to CSV — configurable separator, optional double-quote wrapping, 32K truncation.
  • 🚫Inline delete — select rows in the result table and delete them directly from the org.
  • ⭐Select * config — define priority fields per object so SELECT * always shows your most important columns first.

What does Select * config do?
When you run SELECT * FROM ObjectName, Calumet replaces * with all fields returned by the describe API — but reorders the result columns so your priority fields always appear first, regardless of the API order.

How to use it: open this dialog, pick an object (e.g. Account), then drag fields into the priority list. On the next SELECT * execution, those columns will be shown in the leftmost positions of the result table.

Priority fields are also highlighted with a blue header (★ marker) in the result table.
