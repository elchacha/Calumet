SOQL Executor — Key Features

  • Multi-tab query editor with monospace dark theme
  • SELECT * expansion with configurable priority fields per object
  • Tooling API mode for metadata queries (ApexClass, CustomObject, etc.)
  • Fast count (COUNT_DISTINCT) without fetching all rows
  • Bulk API v2 mode for large datasets (> 10 000 rows)
  • Relation field resolution (e.g. RecordType.Name, Owner.Name)
  • Row filter on result table (client-side search)
  • CSV export with configurable column separator
  • Saved query library with open-folder shortcut
  • Per-tab timeout and max-rows override
