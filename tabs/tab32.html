★ Key features

  • Lists all CronTrigger scheduled jobs from the org via Salesforce SOQL
  • Displays job name, submitter, start date, next fire time, type, target class/flow name, namespace, cron expression (raw and human-readable), and trigger count
  • Filter by job type: Apex Class, Batch Job, Schedule Flow, Dashboard, Report, Report Notification, Others
  • Live text filter — type to narrow results instantly (debounced 500 ms)
  • Kill schedule selected — calls System.abortJob() via anonymous Apex for every selected row, then refreshes the list
  • Double-click any Flow row to open it directly in Flow Builder in the browser
  • Columns auto-resize after each search for optimal readability
