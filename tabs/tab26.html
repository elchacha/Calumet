★ Key features

  • Compares the local deployment package against the metadata currently live in the target org
  • Shows per-component Insert / Update / Delete counts so every change is visible before deploy
  • Flags new components (not yet present in the target) in a dedicated "New" column
  • Retrieves and displays the last modifier and modification date from the org for every component
  • Double-click any row to open an inline diff viewer; CustomField picklists open a dedicated picklist comparator
  • Layout rows open a structured layout diff popup to compare section-by-section
  • Checkboxes let you hide unchanged, changed, or new items to focus only on what matters
  • "Reuse Current backup" re-runs the diff against the cached backup without re-fetching from the org
  • Profile rows are hidden by default and shown only when they contain actual updates
  • In-app notification panel slides in from the top-right to report success or failure at the end of the operation
