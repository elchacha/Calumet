★ Key features

  • Compares the local deployment package against metadata currently live in the target org
  • Shows per-component Insert / Update / Delete counts with color-coded badges before deploy
  • Flags brand-new components (not yet in the target) in a dedicated New badge column
  • Retrieves and displays the last modifier and modification date from the org for each component
  • Double-click any changed row to open an inline side-by-side diff viewer with line-level highlights
  • CustomField picklist rows open a dedicated picklist comparator popup
  • Layout rows open a structured layout diff popup to compare section by section
  • Filter checkboxes let you hide unchanged, changed, or new items to focus only on what matters
  • "Reuse Current backup" re-runs the diff from the cached backup without re-fetching from the org
  • Profile rows hidden by default; shown only when they contain actual updates
  • In-app GemsFX notification slides in from the top-right to report success or failure
  • "Generate AI Prompt" assembles changed file content or paths into a ready-to-paste LLM prompt for functional and technical review
