★ Key features

  • Analyzes the last 90 days of user activity to compute a numerical usage score for each Profile and Permission Set
  • Automatically classifies each item as Keep, Review, or Safe to Remove based on score thresholds
  • Row background colors (green / orange / red) provide an instant visual overview of security health
  • Supports all three security entity types: Profile, Permission Set, and Permission Set Group
  • Real-time filtering by type combo, free-text search (name or description), and recommendation checkboxes
  • "Custom only" checkbox hides standard Salesforce system profiles from the analysis
  • Analysis Details column shows emoji-annotated summaries: active user count, activity level, last login recency, and cleanup justification
  • Summary bar shows total item count, breakdown by type, and overall cleanup potential percentage
  • Export to Excel preserves the current filter state — only visible rows are exported
  • Reuses user activity data shared with the User Activity tab (Tab 63) to avoid redundant API calls
