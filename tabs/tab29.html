★ Key features

  • Query the Salesforce Setup Audit Trail for all profile permission changes within a date range
  • Show field-level security, Apex class, object permission, and system permission changes side-by-side with old and new values
  • Filter results by any keyword or multiple semicolon-separated criteria (e.g. Field-Level;Admin)
  • Toggle "Display full history" to retrieve changes beyond the standard 6-month audit trail window
  • Select rows and add changed profiles directly to the Retrieve list for deployment
  • Save a filtered set of profile changes as a deployable ZIP package into the current project
  • Remove individual rows from the results locally using Delete / Backspace without re-querying
