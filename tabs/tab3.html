★ Key features

  • Deploy, validate, and test-and-deploy metadata packages to Salesforce via the Metadata API
  • Run specific test classes before deployment, or run all local tests for production orgs
  • Quick Check scans package members and shows who last modified each item and when
  • Package tab displays a full tree of all metadata types and members to be deployed
  • Automatic backup injection before deployment and post-deploy information display
  • Profile management: filter deployable profiles per environment with AUT / NAT / FBD status
  • Api Version scanner flags classes and triggers whose API version is more than 5 versions below current
  • Zip package selector lets you load a previously-retrieved deployment package from a project folder
  • Environment-specific overrides (Specif tab) automatically applied at deployment time
  • Missing files detection: identifies package.xml members that have no corresponding file on disk
