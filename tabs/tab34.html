★ Key features

  • Full inventory of org flows with version, status, process type, trigger, and last modifier
  • Filter by type: Screen Flow, AutoLaunched, Workflow, Trigger, Routing, Invokable
  • Active-only toggle to hide inactive or obsolete flows
  • Free-text search across flow name and label, or deep search inside flow XML metadata
  • Double-click (left button) opens the flow directly in Salesforce Flow Builder
  • Double-click (right button) opens the flow's Setup configuration page in the browser
  • Invokable tab maps Apex invokable methods to the flows that call them
  • SubFlows tab reveals parent-child flow dependencies across the entire org
  • Delete obsolete flow versions from the Flows table using the Delete key (with confirmation dialog; blocked on production unless enabled in global settings)
