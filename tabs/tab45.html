Double-click a row to open the WebLink in Salesforce Setup. Unused weblinks are those not referenced by any page layout (Custom Buttons, Lightning Actions, Related List Buttons, or Custom Links sections).

★ Key features

  • Queries all WebLink metadata (Custom Buttons & Links) in the org, excluding managed packages.
  • Cross-references every layout via the Tooling API cache to detect links that appear in no layout — Custom Buttons, Lightning Actions, Related List Buttons, and Custom Link sections are all checked.
  • Flags CustomPageItem weblinks with a warning, as they may still be in use on Home Page layouts even when not found in standard page layouts.
  • Live filter field with debounce: type one or more semicolon-separated keywords to narrow down the results across all columns instantly.
  • Double-click any row to open the WebLink directly in Salesforce Setup via the browser.
  • Helps enforce org hygiene by surfacing obsolete buttons and links that can be safely deleted to reduce metadata clutter.
