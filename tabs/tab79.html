★ Key features

  • Display queries every sandbox of the connected production org via the SandboxInfo Tooling API (name, license type, description, last modified, activation group, source sandbox).
  • New Org creates a Calumet org config pre-filled with the selected sandbox name and a derived username (base.sandboxname) — opens the existing config in edit mode if one already matches.
  • Staleness indicators on the last modified date: ⚠️ over 6 months, 🔥 over 1 year, ❗💀 over 2 years.
  • Color-coded badges assign a unique color per Last Modified By user, activation group and source sandbox; rows are tinted to match the editing user.
  • Live filter across all columns (sandbox name, description, user, group, source) — contains, case-insensitive, debounced.
  • Only available on a production org — a warning is shown otherwise.
