★ Key features

  • Detects page layouts that are not assigned to any profile/permission set at all (Layout is not assigned)
  • Detects layouts assigned only to inactive profiles (profiles with no active users)
  • Detects layouts linked only to inactive record types, making them unreachable at runtime
  • Only profiles with users filter: ignore profiles that have no active users when determining whether an assignment counts as "active"
  • Only active record type filter: restrict the scan to layouts tied to active record types only
  • Double-click any row to open the relevant Object Manager page in Salesforce Setup directly in your browser
  • Results include the object name, layout name, creation date, and a human-readable reason why the layout is considered unused
