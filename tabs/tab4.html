★ Key features

  • Extract all metadata modified in the org between two dates with optional time precision
  • Filter by user — restrict results to your own modifications or a specific named user
  • Exclude managed-package metadata to focus on custom development only
  • Conflict detection — highlights items also modified by other users in the same period
  • Profile change tracking — records old/new values for profile field and object permissions
  • Layout assignment tracking — detects page layout reassignments per profile and record type
  • Export results to a CSV file; re-open the file at any time with "Open csv file"
  • Interactive metadata chooser popup to include/exclude specific metadata types
  • Cache support — skip API calls and reuse a previously fetched metadata snapshot
  • Detected modifications feed directly into the Retrieve tab package.xml
