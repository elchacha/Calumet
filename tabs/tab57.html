★ Key features

  • Parses every Apex class in the org using the Tooling API to extract all method signatures.
  • Detects methods that are never referenced anywhere in the codebase (private dead code).
  • Highlights truly private unused methods in red (PRIVATE METHOD !!!) for immediate attention.
  • Identifies method types: AuraEnabled, Invocable, Controller, Trigger handler, and plain methods.
  • Captures Apex annotations (@AuraEnabled, @InvocableMethod, @TestVisible, etc.) and implemented interfaces per method.
  • Optional hide false public method filter removes methods that appear unused only because they are called from test classes.
  • Optional recompile flag forces a full Apex recompile before analysis to guarantee accuracy.
  • Live text filter narrows results instantly — search by class name, method name, or annotation.
  • Double-click any row to open the corresponding Apex class body in VS Code (or a local temp file) at the exact line number.
  • Requires all classes to compile cleanly — a compile error in any class will abort the analysis.
