★ Key features

  • Search an object and discover every dependent picklist (controlling → dependent pairs).
  • Select one, several, or all picklists via checkboxes before extracting.
  • Per-field result tabs showing controlling values (rows) against dependent options (columns).
  • Controlling values actually used are highlighted in bold.
  • Export the matrix to Excel (.xls) or CSV.
  • Round-trip: edit the Excel file and re-import to update dependencies (single field only).
