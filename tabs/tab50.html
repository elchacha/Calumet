★ Key features

  • Saves daily .hist snapshots of all SObject record counts and storage sizes to disk
  • Select one snapshot to view totals; select two to compare them side-by-side
  • diff and evol(%) columns are color-coded: red = record growth, green = reduction
  • Filter bar narrows the table to matching object names or labels in real time (500 ms debounce)
  • "Harmonize Values" reconciles counts across snapshots when objects were added or renamed
  • "Refresh Daily" forces an immediate snapshot refresh outside the scheduled window
  • Evolution sub-tab renders an interactive Chart.js line chart over any custom date range
  • Object selector sidebar lets you overlay any combination of SObjects on the chart
  • "Group by N days" aggregates data points to smooth daily noise into weekly or monthly trends
  • "Open in popup" opens the current chart selection in a detached floating window
  • Monitoring scope is configurable per org via Calumet Config — unmonitored orgs see a notice
  • Date range is persisted between sessions via memoryPath.properties
