★ Key features

  • Project browser: list all backup ZIP projects stored for the current org, filtered in real time by name, author or date.
  • Metadata search: type a metadata name (case-sensitive) to instantly narrow the list to projects that contain that component.
  • Package contents tree: selecting a project expands its package.xml as a collapsible tree grouped by metadata type.
  • Version diff: double-click any member in the tree to open a side-by-side diff between the backup file and the current deployed file.
  • CustomField diff: for picklist fields, a dedicated popup shows structured field configuration differences.
  • Context menu actions: right-click a project to retrieve the ZIP, export package.xml, unzip for deploy, rename, rollback, or delete the backup.
  • Unzip to deploy: extract a backup directly into a target org's deploy folder, with optional cleanup of the backup sub-folder.
  • Comment tracking: attach and update free-text comments to any backup project for traceability.
  • Class/trigger history (Display tab): search a class or trigger name across all projects to see its full change timeline with Hist count and project-status indicator.
  • Double-click to detail: in the Display tab, double-click a row to open the full change history popup for that component.
