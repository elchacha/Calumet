★ Key features

  • Queries the Salesforce Tooling API to retrieve every custom field definition along with its populated-record count.
  • Displays Usage (raw count of records with a non-null value) and Percent (relative to total records of that object).
  • Formula fields can be hidden with one checkbox click, keeping the list focused on data fields only.
  • Live filter field accepts plain text or multiple keywords separated by semicolons (;) to narrow results instantly.
  • Double-clicking any row opens the corresponding field's setup page directly in the Salesforce browser.
  • Zero-usage fields are immediately identifiable, making them prime candidates for technical debt cleanup and org simplification.
  • Auto-resizes all columns after search to fit the longest value in each column.
