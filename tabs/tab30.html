★ Key features

  • Query field-level security (FLS) for any SObject by profile name, user name, or custom field API name
  • Visual R / W indicators (green check / red cross) show Read and Write access at a glance for every field
  • Real-time Filter bar narrows FLS results by field name, label, profile, or permission set without re-querying
  • User tab aggregates system permissions, object permissions, and field permissions from all permission sets and profiles in a single search
  • Result sub-tabs labeled with live counts after each search — e.g. system(102), objects(4), fields(18)
  • Fields sub-tab checkboxes independently filter by Edit-only, Permission Set source, or Profile source
  • Double-click any row in the Objects or Fields table to open the corresponding Salesforce Setup page in your browser
  • AutoComplete on Object, User, Profile, and Field inputs for fast lookup without memorising API names
