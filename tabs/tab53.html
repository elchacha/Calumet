Custom Metadata Updater

Tab 53

Browse, edit, and mass-upsert records of any Custom Metadata Type directly from an Excel file. Select a __mdt type, display all its records in a dynamic table, export them to an .xlsx workbook, edit values offline, then push them back with a single Upsert. Supports row-level deletion and filtered exports from the current selection.

Key features
Autocomplete type selector
The type field uses an AutoCompleteTextField populated with all __mdt types found in the connected org. Selecting a type immediately triggers a Display query.
Live row filtering
The filter field applies a 500 ms debounce and searches all column values case-insensitively. Separate multiple terms with ; for AND-style narrowing. The record count label updates in real time.
Excel-based upsert workflow
Export all records (or just a template), edit values in Excel, then use Upsert Values to push changes back. The Metadata API processes all rows and reports per-record success or issues in an alert dialog.
Context-sensitive buttons
Delete Selection and Export Selection appear only when one or more rows are selected. DEL and Backspace also trigger deletion from the keyboard.
Direct Salesforce navigation
Double-clicking any row resolves the record's Id from the org and opens the Lightning Setup page for that specific record in the default browser — no copy-paste required.
Dynamic column layout
Columns are generated at runtime from the metadata type's field list. Every custom field is included. Column widths auto-resize to content after each display, and the table supports multi-row selection with Ctrl+C copy.
