Calumet

v9.2 — Salesforce Administration Suite

Calumet is a desktop application for Salesforce administrators and developers. It provides a unified interface for:

• Metadata management — retrieve, deploy, compare and roll back metadata across orgs
• Apex & testing — code coverage, test suites, debug log analysis, dead code detection
• Security & access — profile analysis, permission sets, sharing rules, field-level access
• Data operations — SOQL explorer, record browser, picklist management
• Org-to-org data migration — export, import and clean records while preserving all relationships
